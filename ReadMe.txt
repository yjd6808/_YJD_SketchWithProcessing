유튜브 영상 내용 요약
1. 홈페이지에서 다운 후 그냥 압축 풀면 설치 끝난다.
1. 환경 변수에 압축 푼 경로 추가
1. cmd에 processing-java 실행해보고 잘실행되는지 체크
1. vs 코드 실행해서 확장에서 Processing Language 설치
1. Ctrl + Shift + P 명령줄 연 후 Processing Create Task File 실행
1. 그다음  다시 명령줄을 열어서 Processing Run Project를 실행하면 실행된다.
근데 이렇게하니까 터미널에 오류라든지 실행 여부라든지 안뜸;
1. 테스크를 직접 실행해주는 방식으로 하니까 잘됨. Tasks: Run Task -> Run Sketch(테스크 이름이 Run Sketch임)

중요: 그리고 파일이름과 파일을 담고 있는 폴더 명이 같아야함.
